## Project Overview

RBook is a simple iOS application developed using SwiftUI.  
The app allows users to explore books, view detailed information, and manage their reading interests.

The main goal of this project is to demonstrate:

- Proper use of mobile development fundamentals  
- Clean and organized code structure  
- API integration  
- Proper handling of UI states  
- Clear separation of concerns  

This project is designed as a small-scale graduation project, focusing on code quality and best practices rather than complexity.

---

## Architecture

The project follows the **MVVM (Model ‚Äì View ‚Äì ViewModel)** architecture.

### Structure:

**View**
- SwiftUI screens (Home, Explore, Details)

**ViewModel**
- Handles business logic and state management

**Model / Data Layer**
- API services  
- Data models  

This structure ensures:

- Clean separation of concerns  
- Maintainable and readable code  
- Scalable architecture

## How to Run the Project

1. Clone the repository  
2. Open the project using Xcode  
3. Run the app on Simulator or a real device  
4. Make sure the internet connection is available  
5. Use the Explore screen to search for books  

 Note: Internet connection is required to fetch book data.
